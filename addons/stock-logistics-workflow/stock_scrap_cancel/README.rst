This module adds the cancel status to the scraps and allows the canceled
ones to be converted into a draft.

.. contents::
   :local:

Usage
=====

To use this module, you need to:

1. Go to **Inventory > Operations > Scrap**
2. Create a record and set any product and set quantity to 1.
3. Go to "Validate" button and confirm.
4. Go to "Cancel" button.
5. The scrap is canceled and the stock move has been canceled and
   deleted.
6. Go to "Set to Draft" button.
7. The scrap is in draft state.
